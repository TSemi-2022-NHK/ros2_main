## CAN ID
| CAN ID |   内容    |                    形式                     |
|:------:|:-------:|:-----------------------------------------:|
| 0x0312 |   加速度   |    [x[0],x[1],y[0],y[1],z[0],z[1],0,0]    |
| 0x0313 |  ジャイロ   |    [x[0],x[1],y[0],y[1],z[0],z[1],0,0]    |
| 0x0314 | クオータ二オン | [x[0],x[1],y[0],y[1],z[0],z[1],w[0],w[1]] |
